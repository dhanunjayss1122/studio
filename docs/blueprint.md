# **App Name**: FlappyBeginner

## Core Features:

- Bird Control: Control the bird's upward movement with 'spacebar' presses.
- Pipe Navigation: Navigate the bird through moderately moving pipes.
- Real-time Scoring: Track and display the player's score in real-time.
- Game Over Screen: Display a game over screen when the bird collides, showing the final score and a restart option.
- Progressive Difficulty: Endless game that increases in difficulty as the game progresses

## Style Guidelines:

- Primary color: Bright yellow (#FFDA63), inspired by the vibrant energy and cheerful nature of classic arcade games.
- Background color: Light green (#C1FFC1), desaturated to about 20% brightness to evoke a friendly landscape feel.
- Accent color: Sky blue (#73C2FB), for pipes and scoring elements.
- Font: 'Space Grotesk', a sans-serif for a techy feel; good for both headlines and small amounts of text
- Implement pixel-art icons reminiscent of classic 8-bit games for a nostalgic and engaging experience.
- Classic side-scrolling layout with a focus on simplicity.
- Use simple animations such as the bird flapping and the pipes moving to give a lively experience without complexity.
